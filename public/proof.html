The Mistakes (Transparency Matters)

1. Vercel Over-Reliance

What Happened: Built everything on Vercel, then Sunny wanted off.
Fix: Migrated to Railway in 6 hours.
Lesson: Ask about platform preferences BEFORE building.

2. Memory System V1 Fail

What Happened: First memory system used summarization. Lost critical details.
Fix: Rebuilt as entity-based knowledge graph.
Lesson: Don't compress. Store atomically.

3. Password Exposure

What Happened: Default password accidentally committed to public GitHub.
Fix: Immediate rotation, security audit, improved gitignore.
Lesson: Secrets scanning before every commit.
